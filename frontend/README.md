# Frontend — Slides UI (Vite + React)

Quick start

```bash
cd frontend
npm install
npm run dev
```

Pages implemented (placeholders):
- /login
- /register
- /dashboard
- /templates
- /ai
- /editor/:id
- /slides
- /assets
- /settings
- /presentation/:id

Recommendations
- Use `axios` to call APIs in `backend`.
- Add authentication context / JWT handling in localStorage.
- Use the images you’ll provide to implement page UI components.
